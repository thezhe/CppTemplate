# CppTemplate
Minimalistic, dependency-free, and out-of-source C++ project template.

# Features:
- GitHub Workflow
- Builds docs
- Versioned linter configs
- Changelog
- CMake and CTest
- GNU AGPLv3
- Naming convention: CamelCase Executables/Classes/Scopes, camelBack everything else
